Sobre o Starter Kit

O Starter Kit foi criado para acelerar o início de qualquer projeto web moderno. Inclui uma estrutura organizada com Sass modular, Pug para componentes reutilizáveis, BrowserSync para desenvolvimento rápido e um sistema de temas totalmente personalizável. A ideia é simples: dar-te uma base sólida, limpa e escalável, para que possas focar-te no que realmente importa — construir funcionalidades e criar experiências.

O fluxo de trabalho foi pensado para ser simples e eficiente: sempre que guardas um ficheiro, o sistema compila o Sass, atualiza os templates Pug, otimiza recursos e recarrega o browser automaticamente. Isto permite-te desenvolver mais rápido, testar alterações em tempo real e manter o código organizado desde o primeiro dia. O Starter Kit não é apenas um ponto de partida — é uma base sólida para qualquer projeto moderno.
